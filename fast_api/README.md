## Directions to run 

**Run the following command in terminal** 

1.
```bash
python3.12 -m venv .venv
```

2.
```bash
source .venv/bin/activate
```

3.
```bash
pip install -r requirements.txt
```

**Ensure you are in the fastapi/ directory**

4.
```bash
python3 app.py  
```
